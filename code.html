Mijn coderings ontwikkeling

Sinds ik in september 2023 ben begonnen aan de opleiding Communicatie en Multimediadesign aan de Hogeschool van Amsterdam, heb ik een enorme groei doorgemaakt in verschillende vaardigheden. Een van de belangrijkste hiervan is coderen, een gebied dat voor mij volledig nieuw was. Mijn reis begon met het eerste vak, Internetstandaarden, waar ik kennismaakte met HTML en CSS, en heeft zich verder ontwikkeld tot het werken met geavanceerdere technieken en frameworks in projecten zoals Front-end Development. Hieronder licht ik mijn ontwikkeling in coderen toe aan de hand van vier projecten.

Project 1: Eerste website html en CSS

In mijn eerste codering vak leerde ik de basis van HTML en CSS door een eenvoudige website te bouwen. De opdracht was om een semantisch correcte site te maken over een onderwerp naar keuze. Hoewel de focus nog niet op vormgeving lag, leerde ik al vroeg hoe CSS de presentatie van een website kan verbeteren. Dit project was mijn eerste stap in de wereld van webontwikkeling en legde een solide basis.

Project 2: Mobiele Website voor een Groen Initiatief

Het volgende project was het ontwerpen en bouwen van een mobiele website voor een duurzaam initiatief in Amsterdam. Ik maakte een gids voor tweedehands kledingwinkels in de stad, gericht op iPhone SE-formaat. Hierbij combineerde ik HTML en CSS met vormgeving, waarbij ik een speelse, gezellige en vintage uitstraling creëerde. Dit project gaf mij de ruimte om zelf te experimenteren en mijn vaardigheden verder uit te breiden.

Project 3: Javascript – Mini Game Ontwikkelen

Dit project markeerde mijn eerste kennismaking met JavaScript. De opdracht was om een mini-game te maken waarin alle interacties met JavaScript moesten worden uitgevoerd. Mijn game, een "huisopknapper", liet gebruikers meubels toevoegen aan een virtuele kamer. Hoewel het eindresultaat simpel was, gaf dit project me een waardevolle eerste ervaring met het schrijven van interactieve code.

Project 4: Front-end – Website Reproductie

In het tweede jaar van mijn studie combineerde ik HTML, CSS en JavaScript om een bestaande website na te maken. Ik koos de PlayStation-website, met de focus op de pagina's van de game The Last of Us. Ik begon met een mobiele versie en koos ervoor om de site responsive te maken, zodat deze op verschillende schermformaten goed functioneert. Dit project gaf mij een diepgaand inzicht in responsive design en de praktische toepassing ervan in moderne webontwikkeling.
